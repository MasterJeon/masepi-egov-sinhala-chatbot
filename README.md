# masepi-egov-sinhala-chatbot
### Offline Sinhala E-Government Chatbot
Masepi is an offline Sinhala chatbot designed to assist users with Sri Lankan government services such as NIC, Passport, Driving License, and Birth Certificate processes.

The system uses a local Large Language Model via Ollama and a Streamlit-based interface, ensuring complete offline functionality.
## Features
- Sinhala input and output support
- Fully offline (no internet required)
- Chat-based interface using Streamlit
- Session-based chat history
- Local knowledge base for accurate responses
## Tech Stack
- Python
- Streamlit
- Ollama
- LLaMA 3.1 (primary model)
- Mistral (fallback model)
